1. Apa fungsi library pandas dalam analisis data?
Pandas digunakan untuk mengolah, membersihkan, dan menganalisis data dalam bentuk tabel (DataFrame) dengan cara yang mudah dan efisien.

2. Apa perbedaan antara matplotlib dan seaborn?
Matplotlib digunakan untuk membuat grafik dasar, sedangkan Seaborn menyediakan grafik yang lebih estetik dan lebih mudah digunakan karena dibangun di atas matplotlib.

3. Mengapa perlu melakukan data cleaning sebelum analisis?
Data cleaning diperlukan untuk menghilangkan error, duplikasi, dan nilai kosong agar hasil analisis lebih akurat dan tidak menyesatkan.
